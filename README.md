# GfxPaint
Qt + OpenGL + scenegraph based indexed image editor.

Badly broken, sporadically abandoned, unusable.

Build requirements:
* Qt 5.15/6.0
* C++17/20
* GCC (maybe other compilers)

Build with Qt Creator/qmake for Windows/Linux.
No unbundled external dependencies.
